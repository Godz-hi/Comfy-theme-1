## Installation
### 🔌 Powercord
* Open a terminal and run the following commands
```sh
cd (powercord dir)/src/Powercord/themes

git clone https://github.com/NYRI4/Comfy-theme
```
*Note: `(powercord dir)` is frequently C:\Users\USERNAME\powercord*
